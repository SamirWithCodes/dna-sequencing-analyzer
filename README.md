# 🧬 Bioinformatics Lab Data Analyzer (DNA Sequencing Tool)

A lightweight and efficient Python tool designed to analyze DNA sequences from laboratory data. It validates inputs, calculates essential biological biomarkers like GC-content, transcribes DNA into RNA, and translates RNA into corresponding protein (amino acid) chains.

## 🔬 Features
* **Sequence Validation:** Automatically checks if the input DNA sequence is valid (contains only A, T, C, G) and handles lower/uppercase issues.
* **GC-Content Analysis:** Calculates the precise GC percentage, a crucial metric in laboratory settings.
* **RNA Transcription:** Transcribes the DNA sequence into its corresponding mRNA strand.
* **Protein Translation:** Decodes the RNA codons into a complete amino acid sequence using standard genetic code dictionaries.
